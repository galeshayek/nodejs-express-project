Introduction

Welcome to the API documentation page. This API provides a set of endpoints for managing user authentication, business cards, and more. It is designed to be robust, scalable, and easy to integrate with your applications.

Features

  • User Authentication: Handle user login, registration, and token validation.
  • Business Card Management: Create, read, update, and delete business cards.
  • Error Handling: Centralized mechanism for managing errors across the application.
  • Validation: Ensure data integrity using Joi-based validation schemas.
  • Logging: Track application activities and errors for debugging and monitoring.
  • Database Management: Efficient database connection and schema management.
